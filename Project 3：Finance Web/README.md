## **3. Finance Category 類別**

### **此類別主要是想提供機器學習比較使用者所選類別進行比較。**

### **提問：請問為何要同樣類別進行比較呢？不同類別不能擺在一起比較嗎？**

### **回答：同類別進行比較出來的值會比較貼近，比如說金融業 會缺少某些值，那難道金融業的股票就不推薦嗎。很顯然不適，所以此網站主要類別還是提供使用者觀看財報或是給機器學習使用**
